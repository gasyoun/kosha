# Positioning — "Logeion for Sanskrit"

The identity in one line: **"Logeion for Sanskrit."** A 12-platform live
survey
([COMPARISON.md](https://github.com/gasyoun/kosha/blob/main/COMPARISON.md))
found the read-only collapse already exists — michaelmeyer.fr serves 41
dictionaries on one page — but the **full composite is an empty slot**:
collapse + morphology + corpus evidence + trilingual glosses + open API +
versioned citability.

The founding formula: *meyer's speed + sanskritdictionary.com's features +
the provenance and corpus evidence neither has*. Ambuda/vidyut is the
strongest modern neighbour — consume vidyut, never re-implement it. CDSL
itself is the substrate, not a competitor.

One program, three tracks:

1. **Improve the source** — csl-orig corrections (Cologne ground truth)
2. **Improve the shared Cologne UI** — upstream, for everyone
3. **This project** — the advanced integration layer and UI

See [[what-is-kosha]] for the overview and [[roadmap]] for the build phases.
